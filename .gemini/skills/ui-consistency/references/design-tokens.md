# Design Tokens: Anthony Leuterio Flagship Theme

Centralized design constants for an investor-grade, high-impact professional real estate brand.

## Colors
- **Primary (Midnight Navy)**: `#0A1128`
  - Use for: Backgrounds, authoritative text, and high-prestige UI tiering.
- **Secondary (Metallic Champagne Gold)**: `#C5A059`
  - Use for: Buttons (gradient), prestige highlights, icons, and metadata.
- **Accent Bronze**: `#8E6F3E`
  - Use for: Statistics and badges on light backgrounds.
- **Surface (Subtle Alabaster)**: `#FBFBFB`
  - Use for: Alternating section backgrounds to maintain depth.
- **Text (Slate Charcoal)**: `#2D3748`
  - Use for: High-readability body text.

## Typography Architecture
- **Primary Font**: `Outfit`, sans-serif
- **Heading 1 (Hero)**: `6rem`, `700` weight, `-0.04em` tracking.
- **Heading 2 (Section)**: `3.5rem`, `700` weight, `-0.03em` tracking.
- **Metadata**: All-caps, `2.5px` letter-spacing, `700` weight.
- **Line Height**: `1.9` (Corporate legibility).

## Spacing & Rhythm
- **Architectural Spacing**: `12rem 0` (Section vertical padding).
- **Container**: `1100px` max-width with `4rem` side-gutter.
- **Gaps**: `6rem` (Primary grids), `4rem` (Secondary grids).
- **Radius**: `20px` (Uniform modern radius).

## Depth & Interactions
- **Ambient Depth**: Layered soft shadows using `rgba(0,0,0,0.03)`.
- **Glassmorphism**: `30px` blur with `rgba(255,255,255,0.85)` background.
- **Kinetic Smoothness**: `cubic-bezier(0.23, 1, 0.32, 1)` for all transitions.
