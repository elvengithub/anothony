# Realtor Project

A simple and efficient property listing application built with vanilla HTML, CSS, and JavaScript. This project allows users to browse property listings, search for specific criteria, and filter results.

## Features

- **Property Listings**: View detailed information about various properties, including high-quality images, prices, and locations.
- **Search & Filter**: Find the perfect home by searching for keywords or filtering by price range.
- **Responsive Design**: Optimized for both desktop and mobile viewing with a clean, modern aesthetic.
- **Dynamic Rendering**: Property listings are dynamically generated using JavaScript.

## Tech Stack

- **HTML5**: Structured content.
- **CSS3**: Modern styling with custom variables, Flexbox, and Grid for a professional look.
- **JavaScript (ES6+)**: Handles property data, filtering logic, and DOM manipulation.

## How to Run

Since this project uses vanilla technologies, no installation or build steps are required.

### 1. Using a Local Server (Recommended)
To avoid potential CORS issues or for a better development experience, it's recommended to use a simple local server.

- **VS Code Extension**: Install "Live Server", then right-click `index.html` and select "Open with Live Server".
- **Python**: Run `python -m http.server` (Python 3) or `python -m SimpleHTTPServer` (Python 2) in the project directory.
- **Node.js**: Use `npx serve` or `npm install -g serve` and then run `serve`.

### 2. Opening Directly
You can also open the project by double-clicking the `index.html` file in your file explorer, which will open it in your default web browser.

## Project Structure

- `index.html`: Main entry point and layout.
- `style.css`: Stylesheet for the application (modern UI/UX).
- `script.js`: Core logic for property data, searching, and filtering.
